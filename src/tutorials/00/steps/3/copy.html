So far, we've only updated text content within a ractive. But we can use the same mustache syntax to update element attributes.

Let's add a style attribute to our <p> element:

<p style='color: {{color}}; font-size: {{size}}em; font-family: {{font}};'>
  {{greeting}} {{name}}!
</p>

Now we just need to add some data:

var ractive = new Ractive({
  el: output,
  template: template,
  data: {
    greeting: 'Hello',
    name: 'world',
    color: 'purple',
    size: 4,
    font: 'Georgia'
  }
});

Execute this code. You should see a large purple 'Hello world!' message written in Georgia.
