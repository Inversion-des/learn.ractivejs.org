That's not very exciting. Let's make our template more templatey – replace the hard-coded text in the top-right box with some variables:

<p>{{greeting}} {{name}}!</p>

Then, add some data to it, by adding a data initialisation option to our code (below right) so that it looks like this:

var ractive = new Ractive({
  el: output,
  template: template,
  data: { greeting: 'Hello', name: 'world' }
});

Execute the code (hit Shift-Enter). It should look exactly as it did before.
